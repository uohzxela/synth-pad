# synth-pad

![Image of synth-pad]

HTML5 Web Audio synthesizer and visualizer

# Run it

1. Clone this repo
2. Change directory to repo
3. `python -m SimpleHTTPServer 8000`
4. Go to localhost:8000 and double click on synth-pad.html
